  • The probability that a positive test correctly indicates disease.1
  • The number of true positive results divided by the total with the disease (true positive + false negative).1
  • true positive / true positive + false negative
  • ( 9/ (9 + 1)) x 100% = 90%
  • There is a 90% probablity the screening test will correctly identify subjects with disease.
  • The higher the sensitivity of a test the lower the rate of false negative results.1
  • The probability that a negative test correctly indicates not having disease.1
  • The number of true negative results divided by the total number without the disease (false positive + true negative).1
  • true negative / false positive + true negative
  • ( 980/ (10 + 980)) x 100% = 98.9%
  • There is a 98.9% probablity the screening test will correctly identify subjects with out disease.
  • The higher the specificity of the test the lower the rate of false positive results.1
  • The probability that a patient with a positive test result really does have the condition for which the test was conducted.1
  • The number of true positive results divided by all that test positive (true positive plus false positive results).1
  • A low positive predictive value indicates that many of the positive results are false positives and it would be necessary to follow up any positive result with a more reliable test
  • The probability that a patient with a negative test result really is free of the condition for which the test was conducted.1
  • The number of true negative results divided by all that test negative (true negative plus false negative results).1
  • A high negative predictive value gives high confidence that a negative result is correct.
  1. Screening for Disease; Boston University School of Public Health; Wayne W. LaMorte, MD, PhD, MPH; http://sphweb.bumc.bu.edu/otlt/MPH-Modules/EP/EP713_Screening/
